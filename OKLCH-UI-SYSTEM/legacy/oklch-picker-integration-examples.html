Method 2: Native HTML + CSS OKLCH (Pure CSS) üé®

Note: Modern browsers support OKLCH in CSS! Convert with JavaScript for display.

Live Demo:

HEX: #3b82f6
OKLCH: oklch(55% 0.12 240)

Code:

<input type="color" id="colorPicker" value="#3b82f6"> <div id="colorDisplay"></div> <style> .color-swatch { /* Browser-native OKLCH support! */ background: oklch(55% 0.12 240); } </style> <script> // Convert HEX to OKLCH approximation const picker = document.getElementById('colorPicker'); picker.addEventListener('input', (e) => { const hex = e.target.value; // Use Chroma.js or custom conversion (see Method 4) console.log('Selected color:', hex); }); </script>

‚úÖ Pros

  • Native browser support
  • No dependencies

‚ùå Cons

Method 4: Chroma.js Library (Best Balance) ‚öñÔ∏è

Recommended! Chroma.js provides full OKLCH support with easy conversions.

Live Demo:

HEX: #3b82f6
RGB: rgb(59, 130, 246)
HSL: hsl(217, 91%, 60%)
OKLCH: oklch(55% 0.12 240)

Code:

<!-- Include Chroma.js from CDN --> <script src="https://cdnjs.cloudflare.com/ajax/libs/chroma-js/2.4.2/chroma.min.js"></script> <input type="color" id="colorPicker" value="#3b82f6"> <div id="colorInfo"></div> <script> const picker = document.getElementById('colorPicker'); const info = document.getElementById('colorInfo'); picker.addEventListener('input', (e) => { const color = chroma(e.target.value); // Get OKLCH values const [l, c, h] = color.oklch(); // Display info.innerHTML = ` HEX: ${color.hex()}<br> RGB: ${color.css()}<br> OKLCH: oklch(${(l*100).toFixed(0)}% ${c.toFixed(2)} ${h.toFixed(0)}) `; }); // Create color from OKLCH const oklchColor = chroma.oklch(0.55, 0.12, 240); console.log('OKLCH to HEX:', oklchColor.hex()); // Generate color palette in OKLCH const palette = chroma.scale(['oklch(30% 0.1 240)', 'oklch(90% 0.1 240)']) .mode('oklch') .colors(5); </script>

‚úÖ Pros

  • Easy OKLCH ‚Üî HEX/RGB/HSL conversion
  • Color scales and palettes in OKLCH
  • Small library (~13KB gzipped)
  • Well-documented and maintained

‚ùå Cons

  • External dependency
  • No built-in UI components
  • Need to build picker interface yourself

<!-- Embed Observable OKLAB Color Wheel --> <iframe width="100%" height="684" frameborder="0" src="https://observablehq.com/embed/@shan/oklab-color-wheel?cells=viewof+color%2Ccolor"> </iframe> <!-- Or load as JavaScript runtime --> <script type="module"> import {Runtime, Inspector} from "https://cdn.jsdelivr.net/npm/@observablehq/runtime@5/dist/runtime.js"; import notebook from "https://api.observablehq.com/@shan/oklab-color-wheel.js"; const runtime = new Runtime(); const main = runtime.module(notebook, Inspector.into(document.body)); </script>

‚úÖ Pros
