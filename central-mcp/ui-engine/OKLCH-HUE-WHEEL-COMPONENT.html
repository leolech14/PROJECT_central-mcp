🎨 OKLCH Hue Wheel

A 2D circular selector for hue in the OKLCH color space. Click or drag to select any hue from 0° to 360°. The wheel shows the hue spectrum at the current lightness and chroma values.

Lightness (L) 0.60
Chroma (C) 0.15
Selected Hue (H) 240°
oklch(0.60 0.15 240)
// Integration Example:
const wheel = document.querySelector('#oklchHueWheel'); wheel.addEventListener('huechange', (e) => { console.log('Hue:', e.detail.hue); console.log('OKLCH:', e.detail.oklch); });
